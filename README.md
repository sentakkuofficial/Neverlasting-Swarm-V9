# NEVERLASTING: ORIGINS

A fast-paced **bullet-heaven survival game** set in the world of **Neverlasting**. Survive the endless black wind of **KuroKaze**, gather companions, unlock powerful bond synergies, defeat legendary bosses, and see how long you can endure.

## 🎮 Play

- **WASD / Arrow Keys** — Move
- **P** — Pause
- **Touch Controls** — Supported on mobile devices
- Attacks are automatic
- Collect embers to level up
- Recruit companions and unlock synergies

## ⚔️ Features

### Playable Fighters
- Ichido — Bronze Chain
- Saara — Copper Shield
- Asaki — Nickel Axe
- Miki — Purple Fire
- Kiyora — Graphite Gun
- Kyo — Knife Storm

### Bond System
Recruit allies who fight alongside you and grow stronger throughout the run.

### Synergies
Combine companions to unlock powerful combo abilities.

### Boss Battles
Fight iconic Neverlasting characters including Vidar, Gein, The Headmaster, The Grandmaster, and more.

### Dynamic Biomes
Travel through:
- Ashen Wastes
- Frozen Tundra
- Crimson Fields
- Sunken Swamp
- Golden Desert
- Verdant Ruin
- Magma Reach
- Storm Wastes
- Hollow Void

## 🚀 Running Locally

Simply open:

    index.html

No installation required.
No build tools.
No frameworks.
No dependencies.

## 🌐 GitHub Pages

1. Upload the files to a GitHub repository.
2. Go to **Settings → Pages**
3. Select **Deploy from Branch**
4. Choose **main** and **/(root)**
5. Save.

Your game will be online in a few minutes.
